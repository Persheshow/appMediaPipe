# Tesi LaTeX

File principale:

```bash
Thesis.tex
```

Per compilare con una distribuzione LaTeX installata:

```bash
pdflatex Thesis.tex
bibtex Thesis
pdflatex Thesis.tex
pdflatex Thesis.tex
```

Da personalizzare prima della consegna:

* nome e cognome in `Thesis.tex`;
* nome del relatore in `Thesis.tex`;
* eventuale correlatore, se presente;
* eventuali risultati sperimentali reali nel Capitolo 4;
* eventuali figure dell'applicazione nella cartella `Figures`.

La bibliografia e' gia' allineata alle citazioni presenti nei capitoli.
